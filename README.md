## 1. 레포지토리 구조

└── 2026_Lunarworkshop_presentation.xlsx (발표 리스트)

## 2. 2026_Lunarworkshop_presentation.xlsx 수정 방법
해당 시트에 다음 컬럼을 정확히 맞춰 입력하세요.  
- id: 세션 아이디, 발표 순서대로 증가  
- day: 발표 날짜, d1 = 8월 25일, d2 = 8월 26일  
- track: t1로 고정  
- start, end: 발표 시작 및 종료 시간, 반드시 '텍스트' 서식으로 지정한 후 값을 입력해야 정상 작동합니다.  
- type: 발표 카테고리 (lunarscience, payload, lunarmission) -> CSS 클래스명으로 지정해둔 것이므로, 반드시 이 중 하나를 선택합니다. 카테고리 추가 시 CSS 파일 또한 수정되어야 합니다. 대소문자에 유의합니다.  
- title, authors, abstract, keywords: 제목, 저자, 초록, 키워드 -> 키워드는 쉼표로 구분되어야 합니다. (e.g. 달탐사,자기이상지역,레이너감마)
